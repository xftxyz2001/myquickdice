# 简易的掷骰子程序

## 打包
```bash
pyinstaller -F -w main.py # 单文件
pyinstaller -D -w main.py
```

## 表达式格式：
- `xdy`：掷x次最大值为y的骰子
- `dx`：掷1次最大值为x的骰子
- `x（数字）`：固定数字x

**以上三种表达式可通过加号+进行连接**

### 示例：
- `1d5` 掷1次最大值为5的骰子
- `4d6` 掷4次最大值为6的骰子
- `d7` 掷1次最大值为7的骰子
- `64` 固定数字64
- `8d3+d6+5` 掷8次最大值为3的骰子+掷1次最大值为6的骰子+固定数字5
- `d9+7+2d5+d6` 掷1次最大值为9的骰子+固定数字7+掷2次最大值为5的骰子+掷1次最大值为6的骰子
